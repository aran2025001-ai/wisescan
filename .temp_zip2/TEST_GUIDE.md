## 功能说明

### 修改前行为

## 技术实现

### 使用的方法
- `window.confirm()` - 弹出系统确认框
- `window.history.back()` - 返回上一页

### 代码实现

```typescript
const handleBackClick = () => {
  const confirmMessage = `对话记录将在退出后清空。
全景风险报告已保存在"我的"历史报告中，可随时查看。

确定退出吗？`
  
  if (window.confirm(confirmMessage)) {
    window.history.back()
  }
}
```

## 兼容性

- ✅ 所有现代浏览器支持 `window.confirm()`
- ✅ 所有现代浏览器支持 `window.history.back()`
- ✅ 响应式设计（手机/平板/桌面）都支持

## 注意事项

1. 确认框为系统原生样式，不可自定义外观
2. 点击"确定"后会立即返回，请确保用户已阅读提示信息
3. 其他页面导航按钮（如新对话按钮）不受影响
